### 水果蔬菜商城✔Java✔SpringBoot✔MySQL✔Redis✔可收费远程调试
## 前后端源码 + SQL脚本 + 软件环境安装包 + 万字文档
【项目采用技术】
架构: B/S、MVC
系统环境：Windows/Mac
开发环境：IDEA + MySQL5.7/8 + Maven3 + JDK1.8 + Redis
技术栈：SpringBoot + MyBatis + FreeMarker + JWT + Redis + 雪花算法
采用M（model）V（view）C（controller）三层体系结构，MySQL数据库作为系统数据储存平台，实现了基于B/S结构的Web系统。
【系统功能模块】
该项目分为用户和管理员2个角色，用户角色的功能：登录（JWT的token验证）、注册（邮箱通知、验证码校验）、浏览商品、修改个人信息（上传图片）、修改密码、评论商品、添加商品到购物车（Redis缓存）、提交订单（邮箱通知）、查看订单、收藏商品等等功能。管理员角色的功能：管理用户信息、管理用户评论信息、管理商品信息、管理订单信息等等功能
